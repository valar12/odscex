# Invoke-odscPlan

## SYNOPSIS
Provides organization-scale OneDrive shortcut management capabilities.

## DESCRIPTION
The **Invoke-odscPlan** command is part of the enterprise management surface for odsc. Use these commands to resolve target users, apply desired shortcut state, run assignment plans, validate permissions, and produce operational reports.
