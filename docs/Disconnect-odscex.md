# Disconnect-odscex

## SYNOPSIS
Disconnects the session for the Microsoft Graph API.

## SYNTAX

```
Disconnect-odscex [<CommonParameters>]
```

## DESCRIPTION
The **Disconnect-odscex** function clears the session for the Microsoft Graph API.

## EXAMPLES

### Example 1: Disconnect
```powershell
PS C:\> Disconnect-odscex
```

This command clears the saved session.
