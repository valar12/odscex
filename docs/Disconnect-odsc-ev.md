# Disconnect-odsc-ev

## SYNOPSIS
Disconnects the session for the Microsoft Graph API.

## SYNTAX

```
Disconnect-odsc-ev [<CommonParameters>]
```

## DESCRIPTION
The **Disconnect-odsc-ev** function clears the session for the Microsoft Graph API.

## EXAMPLES

### Example 1: Disconnect
```powershell
PS C:\> Disconnect-odsc-ev
```

This command clears the saved session.
